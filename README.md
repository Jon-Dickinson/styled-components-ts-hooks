## Created using

```Bash
npx create-react-app my-ts-app --template typescript
```

## Emotion was added (styled components)

```Bash
yarn add @emotion/react @emotion/styled
yarn add @babel/plugin-proposal-private-property-in-object
```

## To install and run

```Bash
yarn
```
```Bash
yarn build
```
```Bash
yarn start
```
